
| ID | Requisito | Descrição |
| :---- | :---- | :---- |
| RF01 | Cadastro de Produtos | Permitir o cadastro de produtos de limpeza com campos como nome, marca, composição química, fragrância, volume, tipo de embalagem e aplicação. |
| RF02 |   Controle de Saída de Estoque  | Deve haver uma função para armazenar a saída de produtos do estoque, com o id do produto, a quantidade que está sendo removida e o nome do responsável pela ação.  |
| RF03 |   Visualização do Estoque Completo  | O sistema deve exibir uma página de estoque geral que liste todos os produtos cadastrados, mostrando seus atributos, o estoque atual e o estoque mínimo configurado, para que os funcionários tenham controle do que está disponível. |
| RF04 |  Emissão de Alertas  | O sistema precisa ter uma área de Alertas que funcione de maneira automática, listando todos os produtos que a quantidade atual esteja abaixo do estoque mínimo definido |
| RF05 |  Rastreamento das Movimentações  | O sistema deve armazenar              automaticamente a data e o nome do responsável por qualquer movimentação de entrada ou saída de estoque.  |
| RF06 | Segurança para o acesso ao Sistema.  | O acesso ao sistema deve ser protegido por uma tela de login, que valide o usuário e senha antes de permitir a navegação e operação das funcionalidades de cadastro e movimentação de estoque.  |
